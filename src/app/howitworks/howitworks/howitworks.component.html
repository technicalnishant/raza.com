How it works
3 Step Simple Calling Process

Step 1

Dial the RAZA access number.

Step 2

Dial 011 + country code + the phone number, followed by #.

To Call within USA, Canada or Caribbean: Dial 1 + Area Code + Number followed by # Key

Step 3

Keep talking

Once you have dialed the destination # Do not press the send button
